✅ 인증 시스템 명령어

/인증설정

관리자 전용

서버에 맞게 인증 시스템의 로그 채널, 지급 역할, 팝업창 텍스트를 커스텀합니다.

옵션 (Options)

  • channel (필수) : 인증 로그가 올라올 채널
  • role (필수) : 인증 완료 시 지급할 역할
  • modal_title (선택) : 팝업창 제목
  • input_label (선택) : 입력칸 이름
  • placeholder (선택) : 입력칸 예시 문구

/인증생성

관리자 전용

설정된 텍스트와 디자인을 바탕으로 유저가 누를 수 있는 [인증하기] 버튼을 생성합니다.

💡 팁: 유저들이 입장하는 채널(예: #입장-인증)에서 이 명령어를 입력하세요.

📩 티켓 시스템 명령어

/티켓설정

관리자 전용

1:1 문의 채널이 생성될 위치와 관리자 권한, 팝업창의 질문 내용을 세팅합니다.

옵션 (Options)

  • category (선택) : 티켓 채널이 생성될 카테고리
  • staff_role (선택) : 티켓을 함께 볼 관리자/스태프 역할
  • modal_title (선택) : 팝업창 제목
  • label1 (선택) : 첫 번째 질문 내용
  • label2 (선택) : 두 번째 질문 내용

/티켓생성

관리자 전용

설정된 텍스트와 디자인을 바탕으로 유저가 누를 수 있는 [문의하기] 버튼을 생성합니다.

💡 팁: 고객센터 채널에 이 명령어를 입력해두면 유저들이 쉽게 문의를 남길 수 있습니다.
